欢迎使用 knighkit.

knighkit 是自动化，模块开发，并支持自动打包，支持远程调试的前端开发框架。集成了一些使用方便的辅助开发工具，例如：可以自动检测模版的变化，自动完成重新编译；可启动webnire远程调试工具，可以方便的调试手机端程序。

How to start?

$ npm install knighkit
