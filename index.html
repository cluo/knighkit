欢迎使用 knighkit.

knighkit 是自动化，模块开发，并支持自动打包，支持远程调试的前端开发框架。目的是减少前端开发过程中的重复工作，使你只关注程序本身，而不是创建文件，合并压缩。

Feature

自动化生成项目结构
集成常用的 jquery underscore 库等
集成开发阶段使用的模块化开发库 seajs
新型的模版组成方式，模版即模块。
集成 jstm 模版管理工具，可以使用任意支持预编译的模版框架，强有力的模版构建能力
支持四种模版： jade，ktemplate，mustache（hogan.js），velecity
支持模版数据处理模块
支持将其他模版作为插件添加
自动生成单独模版的测试页面，自动打包数据，方便的测试每个小模块
可定制 html 模版的生成模块
自动化监测模版文件变化，自动重新编译已修改的模版文件
内建预览服务器——不需要启动自己的 HTTP 服务器，内置的服务器用一条命令就可以启动
内建webnire远程调试服务器——一条命令就可以启动，简单那方便的调试你的移动端
支持 css 的合并压缩
内置打包工具，打包后不依赖 seajs
